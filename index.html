Ferramentas

ZABBIX

O Zabbix é uma plataforma de monitoramento de código aberto que permite o acompanhamento em tempo real do desempenho e disponibilidade de servidores, dispositivos de rede, aplicações e serviços. Ele oferece alertas configuráveis, visualizações detalhadas e relatórios que ajudam as equipes de TI a identificar e resolver problemas de forma proativa.

GRAFANA

O Grafana é uma ferramenta de código aberto para visualização de dados e monitoramento, que permite a criação de dashboards interativos e personalizáveis. Ele é amplamente utilizado para exibir dados de diferentes fontes, incluindo bancos de dados de séries temporais como Prometheus, InfluxDB, e o próprio Zabbix. Com o Grafana, é possível acompanhar métricas de desempenho e identificar tendências de maneira visual e intuitiva.

GLP

Solução de código aberto para gestão de ativos de TI e service desk. Ele ajuda a gerenciar inventários de hardware e software, acompanhar solicitações de suporte, gerenciar mudanças e manter a documentação centralizada. O GLPI é amplamente utilizado para melhorar a eficiência do suporte técnico e garantir a rastreabilidade e controle sobre os ativos de TI.
